
# Vyuha - M2

Vyuha is an advanced trading journal and analytics dashboard designed for disciplined traders.

## Features
- **Dashboard**: Real-time overview of performance and habits.
- **Playbook**: Define strategies, rules, and risk management protocols.
- **Analytics**: Deep dive into trade performance with interactive charts.
- **Journal**: Log daily/weekly reflections.
- **Habit Tracker**: Monitor daily routines.
- **Portfolio**: Manage multiple accounts and capital.
- **Data Lab**: (Coming Soon) Backtesting environment.
- **Guru Ji**: (Coming Soon) AI Trading Coach.

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Recharts (Visualization)
- FontAwesome (Icons)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5173

## Project Structure
- `src/components`: UI Components
- `src/types.ts`: Type definitions
- `src/App.tsx`: Main application state & layout
